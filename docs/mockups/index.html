ToDoApp Design Mockups

User-centered, accessible UI/UX mockups demonstrating all features and interactions of the ToDoApp. All mockups are self-contained HTML files with Tailwind CSS and meet WCAG 2.1 AA accessibility standards.

About These Mockups

  • Responsive design (mobile-first, 320px+)

  • Realistic content (no Lorem Ipsum)
  • Edge cases included (empty states, errors, loading)
